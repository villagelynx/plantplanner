Admin Controls

You are managing shared plant photos from the secure admin library. Shared uploads replace the live image source for everyone who visits Plant Planner.

Save Target

Choose This Browser Only for personal testing, or Shared Library to publish the image for everyone.

Each card shows the best image source it already knows about when available: a saved browser override, a shared admin image, a project image, or a wired remote image. In This Browser Only mode, you can paste, drag a photo onto the card, or use Choose File. In Shared Library mode, use Choose File or drag and drop so the upload is saved for everyone. Delete Current Image clears the preview into a replacement-ready slot, and Restore Wired Image brings back the shared, project, or remote preview when available. For permanent project photos, place files in images/plants/ using names like blueberry.jpg or vaccinium-corymbosum.jpg.

Plant Planner's admin image library keeps shared plant photos, project files, and fallback sources organized in one place.
